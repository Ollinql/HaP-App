# Handball Training Planner

Taktiktafel und Trainingsplaner für Handball, gebaut mit React + TypeScript.

## Installation als App (iOS)

1. App-URL in **Safari** öffnen: `https://<dein-username>.github.io/HaP-App/`
2. Unten auf das **Teilen-Symbol** tippen (Quadrat mit Pfeil nach oben)
3. **"Zum Home-Bildschirm"** auswählen
4. Namen bestätigen → **Hinzufügen**

Die App läuft danach vollständig offline und ohne Browser-Leiste.

## Entwicklung

```bash
npm install
npm run dev      # Entwicklungsserver auf http://localhost:5173
npm run build    # Produktions-Build nach dist/
npm run preview  # Build lokal unter http://localhost:4173/HaP-App/ testen
```
