# Handball Training Planner

**Eine App für Handball-Trainer, die ihre Arbeit ernst nehmen.**

Plane Trainingseinheiten, verwalte deine Übungen und behalte den Überblick über die gesamte Saison — alles in einer App, die sich wie eine native App auf deinem iPhone oder iPad anfühlt.

---

## Was kann die App?

- **Dashboard** — Deine Trainingsübersicht auf einen Blick
- **Trainingseinheiten** — Einheiten erstellen, strukturieren und speichern
- **Übungsarchiv** — Eigene Übungen anlegen und wiederverwenden
- **Saisonplaner** — Den Trainingskalender über die gesamte Saison im Blick behalten
- **Taktiktafel** — Spielzüge und Aufstellungen direkt auf dem Feld zeichnen

Alle Daten bleiben **auf deinem Gerät** — keine Registrierung, kein Account, kein Cloud-Zwang.

---

## Installation auf iPhone / iPad

1. **Safari** öffnen und zur App-URL navigieren:
   `https://ollinql.github.io/HaP-App/`
2. Unten auf das **Teilen-Symbol** tippen
3. **"Zum Home-Bildschirm"** auswählen
4. Namen bestätigen → **Hinzufügen**

Die App erscheint wie jede andere App auf deinem Homescreen und funktioniert auch **ohne Internetverbindung**.

---
